# 3ds-upload

Webserver to upload images from your 3DS/Wii U console.

## Setup

In order to use this, you will need Flask and Colorama.

Set `config.py` to your liking and run `py app.py`!

### Windows Copy-Paste

    py -m venv venv
    cd venv
    cd scripts
    activate
    cd ..
    cd ..
    py -m pip install flask colorama
    py app.py
